# İkinci Beyin (Second Brain) Nedir

İkinci beyin, dijital bir bilgi yönetim sistemidir. Beynimizin sınırlamalarını aşmak için tasarlanmıştır.

## Neden Gerekli?

### Beynin Sınırlamaları
- Kısa süreli bellek sınırlı (7±2 öğe)
- Uzun süreli bellek güvenilmez
- Bilgi kaybı kaçınılmaz
- Çoklu görev zorlaştırır

### İkinci Beynin Çözümü
- Sınırsız depolama
- Mükemmel hatırlama
- Aranabilir
- Bağlantılı

## CODE Framework

Tiago Forte'nin ikinci beyin metodolojisi:

1. **Capture**: Önemli bilgileri yakala
2. **Organize**: PARA ile düzenle
3. **Distill**: Progressive Summarization ile özetle
4. **Express**: Bilgiyi kullan ve paylaş

## Hedef

Beynini **düşünmek** için kullan, **hatırlamak** için değil.
